# Snake game

<p> This is the famous game of snake where the goal is to not let the snake die.
  <br>
  The snake can only die when :-
  <ul>
    <li> The snake bites its own body, or</li>
    <li> The snake collides with the boders of the game. </li>
  </ul>
  </p>
